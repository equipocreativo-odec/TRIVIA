
# Trivia ODEC

Juego web de **trivia** sobre la ODEC (estático, sin dependencias). Visual inspirado en escenarios culturales nocturnos, con variables de color para adaptarlo a la identidad del sitio oficial.

## Demo local
1. Abre `index.html` en tu navegador.

## Construcción
- **Stack:** HTML + CSS + JavaScript vanilla.
- **Accesibilidad:** controles con teclado, `aria-live` en mensajes, contraste alto.

## Personalización rápida
- Colores: edita variables `--odec-*` en `styles.css`.
- Tiempo por pregunta: cambia `timePerQuestion` en `app.js`.
- Número de preguntas: ajusta `state.total` en `app.js`.
- Contenido: agrega/edita preguntas en `questions.js`.

## Despliegue en GitHub Pages
1. Sube todo a un repo público (por ejemplo, `trivia-odec`).
2. En **Settings → Pages**, elige **Deploy from a branch** y rama `main`.
3. Comparte la URL `https://TU-USUARIO.github.io/trivia-odec`.
